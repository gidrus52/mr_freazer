# Система дизайна

## Обзор

Создана современная система дизайна для приложения с улучшенными стилями, анимациями и полной поддержкой мобильных устройств.

## Структура

```
frontend/src/assets/style/design/
├── _variables.scss      # Переменные (цвета, типографика, отступы)
├── _mixins.scss         # Переиспользуемые миксины
├── _main-page.scss      # Стили главной страницы
├── _app-view.scss       # Стили AppView
├── _production-view.scss # Стили ProductionView
└── _index.scss          # Главный файл
```

## Цветовая палитра

### Основные цвета
- **Primary**: `#4dabf7` - основной синий цвет
- **Secondary**: `#ff6b6b` - акцентный красный
- **Success**: `#51cf66` - зеленый для успешных действий
- **Warning**: `#ffd43b` - желтый для предупреждений
- **Danger**: `#ff6b6b` - красный для ошибок

### Фоновые цвета
- **Primary**: `#1a1a1a` - основной темный фон
- **Secondary**: `#2d2d2d` - вторичный фон
- **Tertiary**: `#404040` - третичный фон
- **Card**: `#2d2d2d` - фон карточек

### Текстовые цвета
- **Primary**: `#ffffff` - основной текст
- **Secondary**: `#e0e0e0` - вторичный текст
- **Tertiary**: `#b0b0b0` - третичный текст
- **Muted**: `#888888` - приглушенный текст

## Компоненты

### Главная страница

#### Заголовок (main-header)
- Темный фон с границей
- Адаптивная верстка для мобильных
- Контактная информация с иконками

#### Hero секция (main-hero)
- Фоновое изображение с затемнением
- Градиентный текст заголовка
- Адаптивная высота

#### Карточки услуг (service-card)
- Эффект карточки с hover
- Иконки с анимацией
- Градиентные эффекты при наведении

### AppView

#### Навигация
- Стеклянный эффект (glassmorphism)
- Плавные анимации появления
- Адаптивное меню для мобильных

#### Секции
- Градиентные фоны
- Адаптивная верстка
- Плавные переходы между секциями

### ProductionView

#### Блоки продукции
- Разные градиентные фоны для каждого блока
- Карусель изображений
- Улучшенные кнопки контактов

## Миксины

### Градиентный текст
```scss
@include gradient-text($gradient);
```

### Стеклянный эффект
```scss
@include glass-effect($opacity, $blur);
```

### Кнопка с градиентом
```scss
@include gradient-button($gradient);
```

### Карточка с эффектом
```scss
@include card-effect;
```

### Анимации
- `@include fade-in` - появление
- `@include scale-in` - масштабирование
- `@include slide-in('left')` - скольжение
- `@include pulse` - пульсация

## Мобильная адаптация

### Брейкпоинты
- **Mobile Small**: до 424px
- **Mobile Medium**: 425px - 767px
- **Tablet**: 768px - 1023px
- **Desktop**: от 1024px

### Адаптивные изменения
- Гибкие сетки (flexbox/grid)
- Адаптивные размеры шрифтов
- Вертикальная компоновка на мобильных
- Оптимизированные отступы
- Улучшенная навигация

## Анимации

### Переходы
- **Fast**: 0.15s
- **Base**: 0.3s
- **Slow**: 0.5s
- **Slower**: 0.8s

### Эффекты
- Hover lift - поднятие при наведении
- Glow shadow - свечение
- Fade in - плавное появление
- Slide in - скольжение

## Использование

### В компонентах
```tsx
<div class="main-page">
  <div class="main-header">
    <div class="main-header-content">
      // Контент
    </div>
  </div>
</div>
```

### В стилях
```scss
.my-component {
  @include card-effect;
  @include hover-lift;
  
  .title {
    @include gradient-text;
  }
}
```
